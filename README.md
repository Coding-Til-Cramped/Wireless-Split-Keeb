# Wireless-Split-Keeb

A low profile, wireless, 60 key ergonomic split keyboard made for long typing or coding sessions controlled by 2 nice nanos and runs on KMK Firmware

# Features

fully wireless connectivity

low profile switches

ergonomic layout
